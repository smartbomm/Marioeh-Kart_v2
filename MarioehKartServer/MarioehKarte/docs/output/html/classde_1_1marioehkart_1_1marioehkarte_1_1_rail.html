Detailed Description

A Rail object represents one lane from a Carrera rail.

All rail types that are actually implemented in the MariööööhhKart_v2 project are defined static. The following rules are decisive for the naming of the rails:

Straights

main name description
straight_standard 345 mm
straight_standard_third 115 mm
straight_standard_fourth 86.3 mm

Curves

| curve | Carrera radius | angle| lane descripter | left/right |

Carrera radius radius in mm
R1 396 mm
R2 594 mm
R3 792 mm
R4 990 mm

| curve_R1_30_ | inner | R1, 30°, left | | | inner_ | R1, 30°, right|
