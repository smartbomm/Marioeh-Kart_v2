\chapter{Debugging}
\hypertarget{_debugging}{}\label{_debugging}\index{Debugging@{Debugging}}
Zur Fehlersuche während der Programmausführung wurden folgende Debugging-\/\+Funktionen implementiert\+:
\begin{DoxyItemize}
\item Debugging über eine Konsole (Telnet oder Seriell)
\item Debugging über die RGB-\/\+LED auf dem Mikrocontroller-\/\+Board
\end{DoxyItemize}\hypertarget{_debugging_autotoc_md9}{}\doxysubsection{\texorpdfstring{Debugging über die Konsole}{Debugging über die Konsole}}\label{_debugging_autotoc_md9}
In der Datei Definitions.\+h sind zwei Makros zur Ausgabe von Debugging-\/\+Informationen definiert\+:
\begin{DoxyItemize}
\item \#\+DEBUG(...) -\/ Ausgabe von reinem Text
\item \#\+DEBUGF(...) -\/ Ausgabe von formatiertem Text analog zu printf
\end{DoxyItemize}

Je nach aktiviertem Debugging-\/\+Modus erfolgt die Ausgabe unterschiedlich\+:
