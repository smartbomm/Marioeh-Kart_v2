\hypertarget{index}{}\label{index}\index{Overview@{Overview}}
This library enables the creation of a digital map from a Carrera Digital 124/132 racetrack. It\textquotesingle{}s needed for the Mariööööhh-\/\+Kart\+\_\+v2 project to translate linear position data to cartesian coordinates.\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Input}{Input}}\label{index_autotoc_md2}
The whole racetrack must be created by hand with the Carrera-\/\+Mapping-\/\+Tool.
\begin{DoxyItemize}
\item 
\end{DoxyItemize}

The map is built out from different track sections. Each track section represents one part of the track but only one lane. In the library all standard Carrera 124 rails are available. For each curve there are four parts available. An inner and an outer curve for each right and left direction. Switches are not taken into account. Pit stops and control unit lane must be represented with a standard straight rail, becaus the have the same measuring.\hypertarget{index_autotoc_md3}{}\doxysubsubsection{\texorpdfstring{Curve syntax}{Curve syntax}}\label{index_autotoc_md3}

\begin{DoxyEnumerate}
\item Before Listing the Track you have to define a reference coordinate system for the racetrack.
\item Open Carrera-\/\+Mapping-\/\+Tool.\+xlsx with Microsoft Excel.
\item Move to page "{}\+Vorlage zur Erstellung CSV"{}.
\item List your predefined sections\+: Each section has to begin with the keyword "{}\+SECTION\+\_\+\+START"{}. Then enter the following parameters\+:
\begin{DoxyItemize}
\item barcode\+: section identifications number
\item amount\+: amount of rails in the section
\item pos\+\_\+x, pos\+\_\+y\+: cartestian coordinates in mm from the start point of the section. In relation to the reference coordinate system of the whole track.
\item rot\+\_\+z\+: rotation in degrees of the start direction in relation to the x-\/axis
\end{DoxyItemize}
\end{DoxyEnumerate}
